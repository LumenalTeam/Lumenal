# 🌑 LUMENAL OSINT FRAMEWORK

Lumenal is a high-speed, asynchronous tool designed for OSINT (Open Source Intelligence) to find user accounts by nickname across various platforms. Developed with a Stealth Edition aesthetic, focusing on speed, anonymity, and a clean terminal UI.

## 🚀 Key Features
* Async Engine: Blazing fast checks across multiple sites using aiohttp.
* Stealth Mode: Automatic User-Agent rotation to bypass basic anti-bot protections.
* Modern UI: Beautiful terminal interface powered by the rich library.
* Clickable Results: All discovered profiles are interactive links (Ctrl+Click to open).

## 🛠 Installation & Usage

1. Clone the repository:
```bash
git clone https://github.com/LumenalTeam/Lumenal.git https://github.com/LumenalTeam/Lumenal.git```

2.Install dependencies:
* pip install -r requirements.txt```

3.Run the tool:
* python LUMENAL.py```
