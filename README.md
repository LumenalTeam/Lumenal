🌑 LUMENAL OSINT Tool
LUMENAL is a lightweight, powerful Open-Source Intelligence (OSINT) framework designed for rapid information gathering. It combines multiple search modules into a single, sleek terminal interface to help security researchers and enthusiasts track digital footprints.
🚀 Features
 * [1] Username Lookup: Search for a specific nickname across 50+ global platforms and social networks.
 * [2] Email Leak Check: Identify if an email address has been compromised in major data breaches.
 * [3] Image Analysis: Quick tool for metadata and reverse image search preparation.
 * [4] IP Tracker: Get detailed geographical and ISP information about any IP address.
 * [5] Domain Scanner: Analyze websites and domains for infrastructure details.
 * [6] Phone Lookup: Extract information from phone numbers using international databases.
 * 🛡️ Stealth Mode: Built-in proxy support and dynamic User-Agent rotation to bypass anti-bot systems.
🛠️ Installation
To run LUMENAL, you need Python 3.8+ installed on your system.
 * Clone the repository:
   git clone https://github.com/LumenalTeam/LUMENAL.git
cd LUMENAL

 * Install dependencies:
   pip install -r requirements.txt

 * Launch the tool:
   python LUMENAL.py
